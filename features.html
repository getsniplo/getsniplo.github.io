Inline Search

Can't remember a trigger? Type // anywhere on the web to instantly bring up the inline search menu. Browse your entire snippet library without leaving your current field.

  • Works in any text field on any website
  • Fuzzy search by trigger or snippet content
  • Keyboard-navigable results

Smart Variables

Inject dynamic content into your snippets using powerful macros. Stop re-typing today's date or the current page URL.

{date} Today's date
{time} Current time
{clipboard} Clipboard contents
{url} Current page URL
{title} Current page title
{name} Custom prompt

  • Target any field with CSS selectors
  • Site-specific rules that activate automatically
  • Fill multiple fields with one trigger
